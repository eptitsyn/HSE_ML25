# Прогнозирование цен автомобилей

Проект по машинному обучению для предсказания стоимости подержанных автомобилей.

## Аналитика

В работе решалась задача прогнозирования цен подержанных автомобилей на датасете из 6019 объектов. После очистки данных (удаление дубликатов ~10%, заполнение пропусков медианой) и анализа корреляций выяснилось, что наибольшее влияние на цену оказывают мощность двигателя (0.62) и год выпуска (0.41). Разбил столбец name на brand/model/version и создал производные признаки вроде power_per_liter, car_age и km_per_year, что дало прирост качества около 7-10%.

Обучил несколько моделей линейной регрессии с регуляризацией (Lasso, Ridge, ElasticNet), лучшей оказалась ElasticNet с R^2=0.61 и 72% прогнозов в пределах 10% ошибки. Основные проблемы — высокая кардинальность категориальных признаков (много уникальных марок/моделей) и необработанные выбросы. Собрал sklearn Pipeline с предобработкой и создал Streamlit-приложение для одиночных и пакетных предсказаний.

## Запуск

```bash
pip install streamlit pandas numpy scikit-learn
streamlit run app.py
```
